# Personal Writing
_A collection of pieces of writing that I'm drafting before publishing on 
[jcarpinelli.dev](jcarpinelli.dev), and other free-to-publish blogging sites._

### Overview 
This repository includes personal writing I'll be working on throughout 2021.
Pieces will likely be provided via Markdown files, and will be organized / 
categorized in subdirectories. The first piece of writing I've included 
is a poem that attempts to connect Maggie Smith's _Good Bones_ with the 
insurrection at the Capital on January 6th. I'm hoping that further 
developing this repository will lead to personal and professional growth
throughout the _new year_. I'm also hoping that exploring topics through
writing will allow me to see current events with new perspective.
